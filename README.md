# 个人静态博客 / 网站

基于 GitHub Pages 搭建的个人静态网站，使用 Jekyll + Yilia 主题。

## 🚀 部署说明
- 仓库地址：https://github.com/xing123987/xing123987.github.io
- 访问地址：https://xing123987.github.io
- 部署分支：`main` 分支根目录

## 🛠 技术栈
- 静态站点生成：Jekyll
- 主题：Yilia（iissnan 开源主题）
- 托管服务：GitHub Pages

## 📝 维护说明
1.  本地修改后，通过 Git 提交到 `main` 分支，GitHub Pages 会自动部署
2.  主题配置文件：`_config.yml`
3.  文章存放目录：`_posts/`（按 `YYYY-MM-DD-标题.md` 格式命名）
